# BioStat-CLI 📊

Um sistema em Python rodando via terminal (CLI) projetado para cadastrar dados de pessoas, validar as entradas do usuário em tempo real e gerar relatórios estatísticos automáticos (médias de idade, altura e peso).

Este projeto é ideal para praticar conceitos de manipulação de listas, estruturas de repetição, tratamento de exceções (`try/except`) e ordenação de dados.

## 🚀 Funcionalidades

- **Cadastro Dinâmico:** Permite cadastrar quantas pessoas você quiser até que o campo de sobrenome seja deixado em branco.
- **Validação de Dados:** - O sobrenome aceita apenas letras.
  - Idade, altura e peso exigem formatos numéricos válidos e tratam erros de digitação sem derrubar o programa.
- **Relatório Ordenado:** Exibe a lista final de cadastrados em ordem alfabética pelo sobrenome.
- **Métricas Estatísticas:** Calcula automaticamente a média de idade, altura e peso do grupo cadastrado.

## 🛠️ Como Executar o Projeto

### Pré-requisitos
Você vai precisar apenas do **Python 3.x** instalado na sua máquina.
